Text Tools

This tool brings together the lightweight text helpers shown in the editor screenshots into a single page.

  • Line tools for sorting, numbering, shuffling, and deduping
  • Clean up helpers for spacing, wrapping, joining, and URL encoding
  • Find and replace with optional regex support
  • Data extraction for emails, URLs, and numbers
