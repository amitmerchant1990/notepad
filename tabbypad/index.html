Tabbypad

As the name suggests, Tabbypad is a dead simple app that let you write multiple notes.

It has following features:

  • Write multiple notes
  • Search across notes
  • Fullscreen notes to write notes easily
  • Grid layout for easy navigation
  • Simple by design
